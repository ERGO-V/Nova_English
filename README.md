# NovaEnglish

NovaEnglish is a dark-themed Flutter vocabulary trainer for Android and Web.

Implemented areas:
- Built-in dictionary learning and review flow
- Custom dictionaries, units, and word editing
- Local SQLite persistence
- JSON import/export
- User profile basics
